    모바일투표시스템
    (M-Voting)     반응형 웹 홈페이지로 언제 어디서나 쉽고 빠르게 시민의 의견을 수렴하고 정책에 반영하기 위한 모바일 투표 시스템입니다.

    주요기능들

    • 01투표 유형별 서비스
      1. 정책투표 : 엠보팅에 접속 가능한 모든 사람에게 공개
      2. 우리끼리 : 일반시민이 직접 투표를 만들고 참여하는 투표
    • 02공개범위 투표 설정 기능
      1. 전체공개 투표 : 엠보팅에 접속 가능한 모든 사람에게 공개
      2. 비공개 투표 : 선거인명부, 비밀번호, QR투표 등으로 참여자 한정
    • 03중복 투표 방지 기능
      1. 투표 참여 시 전화번호 인증으로 중복 투표 방지 기능 제공

    통합관리자시스템
    (NT-CMS)     사이트 관리, 콘텐츠 관리, 코드 및 권한관리 등 국내 응용 프로그램 환경에 필요한 서비스를 제공하는 통합 관리 시스템입니다.

    주요기능들

    • 01통합 사용자 관리 기능
      1. 시스템 내 모든 회원 일괄 관리 기능
      2. 담당 관리자 등록, 수정, 삭제, 비밀번호 초기화 기능 제공
    • 02통합 콘텐츠 관리 기능
      1. 메뉴, 페이지 등에 대한 각 사이트별 콘텐츠 관리 기능
      2. 팝업, 배너, 사진, 퀵메뉴 관리 및 디바이스별 이미지 등록 가능
    • 03통합 사이트 관리 기능
      1. 각 사이트별 메뉴 및 관리자 권한 기능 제공
      2. 홈페이지 통계를 통해 만족도 현황, PV, UV 등 메뉴별 통계 확인 가능
    • RESPONSIVE
    프로젝트관리시스템
    (PMS)     사업계획, 일정, 산출물, 리크스, 예산 등 프로젝트의 전반적인 업무를 One-Stop으로 통합 관리하는 시스템입니다.

    주요기능들

    • 01프로젝트 관리 기능
      1. PMBOK 기준 9개 영역 포함
      2. 프로젝트 범위, 인력, 일정, 리스크, 예산 등 관리 기능
      3. 각 서비스별 등록, 수정, 삭제 기능 제공
    • 02사용자 관리 기능
      1. 이해관계자 간 소통을 위해 단독 및 그룹 메시지 기능 제공
    • 03권한 기능
      1. 사용자별 권한 관리 기능 제공
      2. 사용자 권한 및 프로젝트/부서 이관을 위한 관리자 페이지 기능 제공
    • WEB
    위반건축물관리시스템 대량의 건축이행강제금 부과업무에 있어 부과 누찰, 착오 등 수작업 부과의 한계상황을 PC로 통합 관리하는 시스템입니다.

    주요기능들

    • 01위반 관련 자료 입력 기능
      1. 위반위치, 적발일자, 적발경로 등에 대한 자료 입력 기능 제공
      2. 정비전·후로 구분하여 현장사진 등록 기능 제공
    • 02이행강제금 자동 산출 서비스 제공
      1. 건축법에 따른 적용요율 입력을 통해 이행강제금에 대한 부과금액 자동 산출 서비스 제공
    • 03위반건축물 관리 기능
      1. 소유자 이력정보, 행정조치 이력현황 정보 제공
      2. 위반건축물 관련 통합 검색 및 검색결과 다운로드 기능 제공
    • 04데이터의 이중 발생 방지
      1. 데이터베이스의 통합으로 데이터의 이중 발생 방지 기능 제공
